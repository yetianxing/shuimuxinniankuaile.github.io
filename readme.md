## 使用
可以修改头像，名称，祝福语，背景视频等等
- 修改头像： `./avatar.jpeg`
- 修改名称： `./index.html` 搜索 小美
- 修改祝福语：在 `./index.js` 搜索 `新年快乐`
- 修改背景视频（video）=> `./video.mp4`
